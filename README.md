🌧️ Bangalore Rainfall & El Niño Prediction (ML Project)
 Overview :

This project analyzes Bangalore rainfall data to predict El Niño events using machine learning models.
It involves data preprocessing, model training, evaluation, and visualization to derive meaningful insights.

Features :

● Data cleaning and handling missing values
● Feature selection using monthly rainfall data
● Binary classification of El Niño events

Model training using:

   ● Logistic Regression
   ● Decision Tree
   ● Random Forest
Model evaluation using:
   ● Accuracy, Precision, Recall, F1-score
   ● Confusion Matrix
   ● ROC-AUC Curve
   ● Cross-validation
   
Visualization of:
   ● Confusion matrices
   ● ROC curves
   ● Feature importance
   
🛠️ Technologies Used
   ● Python
   ● Pandas, NumPy
   ● Matplotlib, Seaborn
   ● Scikit-learn
Dataset
Bangalore rainfall dataset (monthly rainfall + El Niño indicator)

⚙️ Workflow :
     1.Load and explore dataset
     2.Handle missing values
     3.Convert El Niño labels to binary
     4.Split data into training and testing sets
     5.Train ML models
     6.Evaluate model performance
     7.Visualize results and compare models
     
✨ Results :
     ● Compared multiple models for prediction performance
     ● Evaluated using ROC-AUC and cross-validation
     ● Identified important rainfall features influencing predictions
     
Future Improvements :

     ● Use advanced models (XGBoost, LSTM)
